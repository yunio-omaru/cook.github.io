# 🍳 ゴリのレシピ (cook.github.io)

シンプルで高速に動作する、自分専用のWebレシピ帳システムです。  
HTMLファイルとJSON（`recipe-list.json`）をGitHub API経由で直接更新し、サーバーレスでレシピの追加・編集・一覧表示を行います。

## ✨ 主な機能

- **超高速な一覧表示**：JSONカタログを読み込み、主食・副食などのジャンルやタグで爆速フィルタリング。
- **直感的なレシピ追加・編集**：ブラウザ上のフォームからレシピを入力するだけで、自動でHTMLを生成。
- **画像埋め込み対応**：料理写真はBase64データとしてHTML/JSONに直接埋め込まれるため、面倒な画像ファイルの管理が不要。
- **GitHub自動保存**：個人アクセストークン（PAT）を使って、ブラウザから直接GitHubリポジトリへコミット＆プッシュ。

## 🚀 使い方（レシピの追加方法）

1. `index.html` をブラウザで開きます（GitHub PagesのURLでも可）。
2. 画面右上の **「＋ レシピ」** ボタンを押します。
3. 初回のみ、一番上の **「GitHub自動保存の設定」** を開き、以下を入力します：
   - **GitHubユーザー名**： `[あなたのGitHubユーザー名]`
   - **個人アクセストークン (PAT)**： GitHubで生成した `ghp_` から始まるトークン
4. レシピのタイトル、材料、作り方を入力し、料理の写真を選択します。
5. **「GitHubへ自動保存」** を押すと、リポジトリに新しいレシピHTMLと、更新された `recipe-list.json` が直接保存されます。
6. 数分待つと、GitHub Pagesにサムネイル付きで自動反映されます。
